Resolving an Issue

 

 

 

If you have been assigned as a resolver for an Issue, you will receive an Alert to notify you the Issue is available for your attention.  The Issue will also display in your Issue panel.

 

  1. Begin at your Homepage.
     

  1. Go to the Issue panel and click Description to open the Issue to be resolved.
     

Note:

A Resolve Issue pop-up window displays.

 

  1. Review the following items:

  • Site Investigator-  The Protocol / Site and Investigator information is displayed.

  • Issue Post Date - The date the Issue was posted.

  • Resolver - The name of the resolver.

  • Previous Resolver - The name of any resolvers to whom this Issue has previously been assigned.

  • Classification - The category to which the Issue has been assigned.

  • Priority - The priority assigned to the Issue, this determines the timeframe for the resolver.

  • Description - Provides details about the Issue.

  • Action - Any suggested actions provided by the originator will be displayed.

 

  1. To close the Issue click one of the following in Issue Status:

Close with Resolution

If you have a resolution, type the resolution in the text box provided.  Click Close with Resolution to change the status of the Issue.

Close without Resolution

It is possible to close an Issue without a resolution.  Type any comments in the Issue Resolution field.  Click Close with out resolution to change the status of the Issue.

Reassign

If the Issue should be reassigned to a different resolver or reviewer, identify the individual it should be reassigned to: :

  1. Identify a resolver to whom the iIssue will be reassigned.

  2. Enter the name of the resolver in the Letter or Name field.

  3. Click Search

  4. Highlight the Resolver’s name in the Search Results box

  5. Click the right arrow adjacent to the Resolver box to move the selected name into the Resolver box.

Note: To clear the Resolver box, click Clear.

 

  1. You can click:

  • Submit to process the Issue with your changes.

  • Close Window to exit without submitting changes and no modifications are recorded.
     

Note:

If you have been assigned as a Reviewer of an Issue, the Resolver Issue pop-up box will display all information related to the Issue.  You will not have access to the close / reassign options and are not responsible for resolving the Issue.  After reviewing all Issue information, click Close Window.
