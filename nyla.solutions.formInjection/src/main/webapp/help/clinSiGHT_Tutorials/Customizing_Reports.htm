Customizing Reports

 

 

Use the following steps to customize clinSIGHT reports.  If you are assigned in IMPACT with multiple roles, you can select one role to customize your clinSIGHT information.
 

  1. Click the Reports tab.

 

  1. Review the Reports tab and Sub-sections:
     

Note:

The Reports and Sub-sections may vary depending on your role.  You will be able to generate reports for the sites and/or personnel for which you have been assigned responsibility.

 

The reports listed below are a sampling of those seen in clinSIGHT.

Visit Targets

Visit Target Report by Protocol, SMN & adjustable date range, in order by SMN last name.  Contains the:

  • Percent of actual visits that adhere to the visit frequency defined in the SMP.

  • Percent of actual 1st monitoring visits that adhere to the defined range in the SMP for FPFV to first Monitoring Visit.

  • Percent of actual site closure visits that adhere to the defined range in the SMP for LPLV to Site Closure.

Visit Date to Report Completion

Listing by Report Type and adjustable date range, in order per SMM by SMN last name.  Contains the:

  • SMM Name

  • SMN name

  • Number of sites

  • Number of visits

  • Average days to report completion

  • Percent in range

SDV Summary

SDV Summary Detail Report for a SMN selected from the SDV Summary report, reported by Protocol/Site.  Contains the:

  • Protocol/Site

  • Subject visits (number of, within the date range)

  • SDV (Yes, No, Partial)

*EDC studies not included in this report.

Randomized Target Achieved

Randomized Target Achieved Detail Report by SMN selected from Randomized Target Achieved report, in order by protocol/site.  Contains the:

  • Protocol/site                      

  • Target

  • Actual

  • Percentage of target

Low-Enrolling Sites

Low-Enrolling Sites Report by Protocol, SMN and adjustable date range. In order by Protocol/Site.  Contains the:

  • Protocol/Site                     

  • SMN name

  • SIV Date

  • Close Date

  • Enrollment

Significant Protocol Deviations

Significant Protocol Deviations Detail Report by Protocol and adjustable date range selected from the Significant Protocol Deviations report. In order by Protocol.  Contains the:

  • Protocol/Site

  • Number of deviations

Site Monitor Workload

Site Monitor Workload Report by SMN in order by SMN last name.  Contains the:

  • SMN name, standard or unblinded

  • Number of protocols

  • Number of sites

  • Number of actual subjects randomized

  • Number of actual subjects planned

Site Monitor Site Visit

Site Monitor Site Visits Report by adjustable date range in order by SMN last name.  Contains the:

  • SMN Name

  • Total Visits

  • Meetings with the Investigator (percent)

 

  1. Click Save to save information entered.
